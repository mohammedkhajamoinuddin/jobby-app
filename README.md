# Jobby App

A modern job portal application built using React.js.

## Features

- Secure Login Authentication using JWT
- Protected Routes
- Job Search
- Employment Type Filters
- Salary Range Filters
- Job Details Page
- Similar Jobs
- Loading & Failure States
- Responsive UI

## Tech Stack

- React.js
- React Router
- JavaScript (ES6)
- CSS3
- REST APIs
- JWT Authentication

## Project Structure

```
src/
 ├── components/
 ├── constants.js
 ├── App.js
 └── index.js
```
